# 背景・前提 / Context
- 言語: Swift (SwiftUI)
- フレーズデータベース: SwiftData（元データであるCSVファイルをインポートして生成）
- 音声データ: 各フレーズごとに音声ファイル（mp3）があり。
- ターゲット: iOS 17以降
- カレントフォルダには、iosアプリのテンプレートが保存されているので、それに変更を加えたり、新しいファイルを追加したりしてアプリを完成させてください。
<br><br>

# データ構造
## フレーズデータCSVファイル
### ファイルパス: csvs/TOEIC_kin_phrases.csv
### ファイル構造: 
|No.|項目名|意味|
|-|-----------|----------------------|
|1|source_file|音声ファイルのベース名|
|2|level|英単語のレベル|
|3|index|連番|
|4|word|英単語|
|5|ja_translation|日本語フレーズ|
|6|phrase|英語フレーズ（フレーズ中の英単語を"[]"で括っている）|
|7|phrase_repeat|インポートしない|
|8|raw_text|インポートしない|

## フレーズデータベース(SwiftData)
|No.|項目名|意味|
|-|-----------|----------------------|
|1|source_file|音声ファイルのベース名|
|2|level|英単語のレベル|
|3|index|連番|
|4|word|英単語|
|5|ja_translation|日本語フレーズ|
|6|phrase|英語フレーズ（フレーズ中の英単語を"[]"で括っている）|
|7|correct_count|連続正解回数（初期値: 0, 正解で+1, 不正解で-1）|
|8|memo|ユーザーが自由に書き込めるメモ|

## 音声データ（ファイル名規則）
音声データファイルは次の位置に保存されているので、アプリ内に取りこむこと。
|音声内容|ファイルパス|備考|
|--------------|---------------------------------------------------------------------------------------|----------|
|英単語|mp3s/[英単語のレベル]/[音声ファイルのベース名]_[連番を100で割った余り（3桁の0埋め）]_parts0.mp3||
|日本語フレーズ|mp3s/[英単語のレベル]/[音声ファイルのベース名]_[連番を100で割った余り（3桁の0埋め）]_parts1.mp3|取り込まない|
|英語フレーズ|mp3s/[英単語のレベル]/[音声ファイルのベース名]_[連番を100で割った余り（3桁の0埋め）]_parts2.mp3||
|ページめくり音|mp3s/others/paging.mp3||
```
	(例)
		英単語レベル: level_730
		音声ファイルのベース名: TOEIC_kin_kaitei_10
		連番: 101
		音声内容: 英語フレーズ
	この場合のファイルパス:
		mp3s/level_730/TOEIC_kin_kaitei_10_001_parts2.mp3
```

# 具体的な要件 / Requirements
- [ ] 基本画面構成: 連番, 英単語レベル, 英単語, 日本語フレーズ, 英語フレーズ, 正解表示ボタン, 正解数, 正解・不正解ボタン, メモ, メモ更新ボタン, 単語・フレーズ検索フィールド, ステータスバー（英単語、日本語フレーズ、英語フレーズの直下に音声再生ボタンを配置。これらのアイテムを効率よく配置し、単語カードのような見た目に）
- [ ] フレーズデータの抽出方法: データベースから、"フレーズデータ抽出ルール"に従い抽出、（英単語レベルを固定して抽出することも可能）
- [ ] フレーズデータ抽出ルール: "連続正解回数"が少ないものを優先的に表示、ただし、一度表示したものは最低でも10問の間をおいた後表示する。
- [ ] テストの方法: "テストの方法_1"または"テストの方法_2"のいずれかでマスキング表示
- [ ] テストの方法_1(英単語テストモード): "英単語"、"英語フレーズ中の[]で囲まれた単語"をマスキングして表示。
- [ ] 英単語テストモードにおけるマスキング方法: 英単語〜単語が何文字であっても"________", 英語フレーズ〜フレーズ中の[]で囲まれた単語について、[]は除去した上で、単語が何文字であっても"________"
- [ ] テストの方法_2(日本語テストモード): "日本語フレーズ"をマスキングして表示。
- [ ] 日本語テストモードにおけるマスキング方法: "日本語フレーズ"全体を"？？？"と置き換えて表示
- [ ] 解答の表示方法: "正解表示ボタン"をタップすることでマスキングされた位置に正解を表示
- [ ] メモ登録機能: "メモ"フィールドにメモを入力し、"メモ更新ボタン"をタップすることでメモを更新することが可能
- [ ] 右スワイプで次の英単語に移動できる
- [ ] 左スワイプで前の英単語に移動できる（表示英単語の履歴を保持できること）
- [ ] ステータスバー: 日単位で、出題数（正解ボタンをタップした数 + 不正解ボタンをタップした数）、正答率（正解ボタンをタップした数 / 出題数）を％で表示, 出題数のクリアのタイミングは午前５時, 毎日の出題数, 正答率をデータベースに登録し、確認できるようにする。
- [ ] データベース修正機能: 日本語フレーズや英語フレーズに誤記がある場合があるので、それぞれのフィールドを長押しすることで、修正画面を表示し、データを修正できるようにする。
- [ ] 英単語, 英語フレーズについて、再生ボタンを押すことにより再生できるようにする。
- [ ] フレーズデータベースのエクスポート（メインテナンス用）: 出力場所を指定して、CSV形式で出力できるようにする。エクスポートボタンは、インポートボタンの隣に配置。
- [x] フレーズデータベースのインポート機能の修正: 出力したエクスポートファイルをインポートできるようにする。
- [x] 正解/不正解ボタンのタップで、触覚フィードバック: 正解ボタン〜振動1回, 不正解ボタン〜振動2回
- [x] 音声再生機能の変更: 英単語, 英語フレーズの再生方法を「mp3ファイルの再生」から「iOSのTTS（Text-to-Speech）機能を使った再生」に変更
- [ ] 右スワイプ,左スワイプでカードをめくった時に、「ページめくり音」を非同期で再生するようにする。
